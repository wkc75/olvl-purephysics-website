**Derived units** <InlineCitation
  label="1c"
  content="express derived units as products or quotients of SI base units, and use the named units listed in the “Summary of Key Quantities, Symbols and Units” as appropriate."
/> are quantites obtained by combining base quantites using physical relationships
(formulae).

**_For example_**

Speed _v_ is a derived quantity.

Speed is derived from base quantites length and time, $v = \frac{s}{t}$.

The units of derived quantites are also derived from base units.

The units of _s_ and _t_ are **m** and **s** respectively, based on the formulae $v = \frac{s}{t}$, the unit of _v_ is given as, **m/s**.

## **1.1.4 Homogenity of Physical Equation**

<InlineCitation
  label="1d"
  content="use SI base units to check the homogeneity of physical equations."
/>

For an physical equation to be plausible, it must be **_homogenous_**,
meaning terms on both sides of the equation must have the same unit when expressed in SI units.

This is because only quantites with the same unit can be **added**, **subtracted** or **equated**.

## **1.1.5 Estimation**

<InlineCitation
  label="1e"
  content="make reasonable estimates of physical quantities included within the syllabus."
/>
**Estimation** involves making **approximate calculations** using simple numbers
and correct **orders of magnitude**.

<br />

**Estimation is used when**:

- Exact values are **unknown or unnecessary**
- Only the **scale** (tens, hundreds, thousands) matters
- Checking whether a calculated answer is **reasonable**

  <br />

### **Key Estimates**

- Visible light wavelength: **≈ 10⁻⁷ m**
- Size of an atom: **≈ 10⁻¹⁰ m**
- Size of a nucleus: **≈ 10⁻¹⁵ m**

  <br />

### **Estimation Strategy**

- Define the **unknown**
- Identify relevant **known values**
- Relate them using simple assumptions
